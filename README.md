# pokedex

## Stack

Para el desarollo de esta webApp tuve la idea de implementar conlas siguientes tecnologias:

- Vue.js 2 y Vuex
- Tailwind.css
- EsLint
- Axios

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
